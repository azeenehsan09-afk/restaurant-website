  # 🍽️ Restaurant Website

A responsive restaurant website built using HTML, CSS, and Bootstrap 5.

## 📖 Overview

This project is a simple multi-page restaurant website created to practice front-end web development concepts. It includes a home page, menu, about page, and contact page while demonstrating responsive design using Bootstrap.

## ✨ Features

- Responsive design using Bootstrap 5
- Home page with featured dishes
- Menu page with food cards and prices
- About page with restaurant information
- Contact page with a responsive contact form
- Clean and simple user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## 📂 Project Structure

```
Restaurant-Website/
│── index.html
│── Menu.html

│── Contact.html

└── README.md
```

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/restaurant-website.git
   ```

2. Open the project folder.

3. Open `index.html` in your web browser.
